# Fansub Workbench
A web-based workbench for fansub groups.

Coded by Huluyou, since May 8, 2023.

v0.0.0.0. Still in development (as of May 2023).

## Frontend

`./client` is the frontend, in Vue.

Dependencies:
* Node.js
* Vue
* axios
* Element UI

## Backend

`./server` is the backend, in Python with Flask.

Dependencies:
* Python
* Flask
* SQLite
